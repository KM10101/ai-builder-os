# 技能孵化清单

这一页用于记录“哪些内容未来可能值得做成 skill”。

当前阶段不急着实现，先记录想法和判断依据。

## 什么时候值得升级成 skill

当一个流程满足下面大部分条件时，可以考虑：

- 你已经实际重复使用过多次
- 不只是 prompt，而是一套明确流程
- 有稳定输入、步骤和输出
- 适合长期复用
- 已经能说清楚适用场景和不适用场景

## 当前候选方向

- 任务分解器：根据任务描述推荐 Claude Code / Codex 工作流
- 案例沉淀器：把一次实践快速整理成 case 文档
- Prompt 提炼器：从多个案例中抽出共性 prompt
- 模式归纳器：从若干案例总结可复用 pattern

## 当前原则

先积累内容，后封装 skill。
