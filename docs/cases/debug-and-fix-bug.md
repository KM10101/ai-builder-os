## 示例填法（参考）

### 问题现象
测试环境中某个接口偶发返回状态异常，只有在订单超时边界附近容易复现。

### 工具选择与分工
- 先用 Claude Code 理解现象、推测原因、定位代码路径
- 再用 Codex 执行修复

### 使用的 Prompt
先不要急着修代码。请帮我分析这个 bug 的可能原因：
- 现象是 XXX
- 复现条件是 XXX
- 我怀疑和超时状态切换有关

请先给出最可能的几个原因、对应代码路径，以及我应该优先排查哪些文件。

### 有效点
- 先让它分析原因，而不是直接修，减少了盲改
- 给出“现象 + 复现条件 + 怀疑方向”后，结果明显更聚焦

### 下次优化
下次可以更早提供日志片段、错误栈或关键状态值，帮助它更快收敛。
