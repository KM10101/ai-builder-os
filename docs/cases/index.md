# 真实案例库

这里记录你在真实工作和学习中，如何使用 Claude Code、Codex 等工具完成任务。

当前建议优先积累下面几类高频案例：

- 读代码 / 理解现有实现
- 小需求实现
- Bug 排查与修复
- 代码 Review
- 重构与清理

## 使用建议

每做完一次真实任务，优先问自己：

- 这次任务属于哪一类？
- 我用了什么工具，为什么？
- 哪些 prompt 有效？
- 哪些地方跑偏了？
- 下次能不能提炼成 pattern / prompt？
