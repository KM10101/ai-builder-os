# AI Builder OS

一个用于学习、实践、复盘 AI 应用开发能力的个人成长操作系统。

## 核心想法

AI Builder OS 希望帮助你：

- 用结构化方式学习理论知识
- 把知识持续转化为实验和项目
- 提升 AI 辅助开发工作流
- 记录过程中的进展、问题和反思
- 积累长期可见的成长证据

## 第一版重点

第一版的重点是先把系统地基搭起来：

- 明确项目愿景
- 明确能力地图
- 明确系统运转循环
- 明确第一阶段路线图
- 提供可复用的实验与复盘模板
