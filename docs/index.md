# AI Builder OS

一个面向 **Claude Code / Codex / AI 编程实践** 的个人操作系统。

这个站点的目标不是单纯做学习计划，而是帮助你把真实工作和学习中的经验，沉淀成可复用的资产：

- **Patterns**：方法论与执行模式
- **Prompts**：可复制的提示词模板
- **Practices**：外部优秀实践的学习记录
- **Reference**：常用命令与速查内容
- **Cases**：真实任务复盘
- **Skills**：未来值得封装的复杂流程

## 当前重点

当前阶段优先做三件事：

1. 把 Claude Code / Codex 的高频用法沉淀下来
2. 把真实工作中的实践整理成案例
3. 建立一套能持续演进的个人知识结构

## 使用建议

推荐你按这个节奏使用这个仓库：

1. **遇到真实任务** → 先查 `patterns` / `prompts`
2. **完成任务后** → 到 `cases` 记录复盘
3. **发现重复打法** → 提炼为 `pattern`
4. **模式足够稳定** → 放进 `skills` 作为待封装能力
