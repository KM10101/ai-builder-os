# Project Structure

## 当前结构

```text
docs/
  .vitepress/
  guide/
  system/
  roadmap/
  reviews/
  experiments/
```

## 结构说明

- `guide/`：说明这个系统怎么使用
- `system/`：定义系统本身的核心设计
- `roadmap/`：存放阶段性学习路线图
- `reviews/`：存放复盘模板以及后续复盘记录
- `experiments/`：存放实验模板以及后续实验记录

## 后续可扩展目录

后面可以继续增加：

- `projects/`：记录项目实现和阶段成果
- `notes/`：记录专题知识笔记
- `playbooks/`：沉淀 AI 编码工作流和可复用流程
- `assets/`：存放图表、截图和其他静态资源
