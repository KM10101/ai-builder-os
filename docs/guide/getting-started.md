# 开始使用

## 这是什么项目

AI Builder OS 不只是一个笔记仓库，也不只是一个代码项目。

它是一个同时用于以下目标的系统：

- 规划学习方向
- 通过实验和项目做实践
- 提升 AI 辅助开发能力
- 记录复盘和能力成长证据

## 建议从这里开始

1. 先阅读项目愿景。
2. 再查看能力地图。
3. 按照第一阶段路线图开始执行。
4. 创建第一条实验记录。
5. 启动第一个小项目。
6. 完成第一次周复盘。

## 基本工作原则

- 用小周期持续学习
- 理论和实现要保持连接
- 每一轮都要产生可复用输出
- 定期复盘，不做无反馈投入
- 先迭代起来，再逐步优化系统
