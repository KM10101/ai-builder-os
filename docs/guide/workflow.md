# Workflow

## 标准学习闭环

每个学习主题都尽量按照下面这个循环推进：

1. 学习核心概念。
2. 做一个最小实验。
3. 放到真实项目语境里应用一次。
4. 记录哪些有效，哪些无效。
5. 提炼成可复用的方法。

## 标准执行节奏

### Weekly

- 明确本周重点主题
- 明确本周实践任务
- 至少完成一个实验或一个功能
- 完成一次周复盘

### Monthly

- 回顾能力成长情况
- 调整路线图优先级
- 判断哪些内容值得深入、暂停或放弃

## AI Tool Usage Principle

AI 编码工具应该被当作能力放大器，而不是替代思考的工具。

重点训练：

- 任务拆解
- 清晰的实现约束表达
- 代码审查与验证
- 迭代式修正
- 可复用的 AI coding workflow
