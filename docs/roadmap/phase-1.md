# Phase 1

## Theme

搭建 AI 应用开发基础能力，以及 AI 辅助开发工作流的初始版本。

## Objectives

- 理解 LLM 应用的基本结构
- 练习在可控流程中使用 AI 编码工具
- 完成至少一个小而完整的 AI 应用
- 建立第一版可重复执行的复盘节奏

## Recommended Topics

### Topic 1: LLM application basics

- API 调用基础
- prompt 角色与结构
- structured output
- function calling / tool calling 基础

### Topic 2: AI-assisted development workflow

- 任务拆解
- 给 coding agent 编写实现说明
- 验证生成代码
- 迭代式调试和修正

### Topic 3: first small application

建议方向：

- 文档总结器
- 笔记整理工具
- 网页内容总结工具
- 简单知识助手

## Exit Criteria

满足以下条件时，可以认为第一阶段完成：

- 一个小应用已经跑起来
- 至少写出两条实验记录
- 至少完成一次周复盘
- AI coding workflow 已有第一版文档沉淀
