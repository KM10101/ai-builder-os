# Claude Code 常用命令速查

> 这是一份个人速查页，优先记录你高频会用到的内容。后续可以持续补充、修正。

## 当前使用建议

Claude Code 更适合优先承担：
- 读代码
- 拆任务
- 给方案
- 做 review

## 常见用法（待你结合实际继续补充）

### 启动 / 进入项目

```bash
claude
```

在项目目录下启动后，优先先让它：
- 理解仓库结构
- 总结关键模块
- 不要直接修改

### 常见起手语句

```text
先不要改代码，先帮我理解这个仓库里和 XXX 相关的实现。
```

```text
先输出修改计划，不要直接动手。
```

```text
请 review 这次改动，优先指出高风险问题。
```

## 推荐使用习惯

- 先让它解释现状，再让它修改
- 先让它出方案，再进入实现
- review 时要求它优先关注高风险问题
- 大任务先拆成更小范围
