# Codex 常用命令速查

> 这是一份个人速查页，先记录高频有用内容，后续按你的实际使用方式持续迭代。

## 当前使用建议

Codex 更适合优先承担：
- 执行明确修改
- 实现小功能
- 修明确 bug
- 补测试

## 常见起手语句

```text
按下面要求修改代码，尽量少改动，不要顺手重构。
```

```text
先列出你准备修改的文件和原因，等我确认后再执行。
```

```text
改完后请总结修改点、风险点，以及建议我执行哪些验证命令。
```

## 推荐使用习惯

- 给 Codex 的目标尽量具体
- 明确限制改动范围
- 要求它不要顺手做无关优化
- 改完后一定让它自查
